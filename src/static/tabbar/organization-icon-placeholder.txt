组织图标占位符说明
==================

需要创建的图标文件:
1. static/tabbar/organization.png - 未选中状态图标(灰色 #999999)
2. static/tabbarSclect/organization.png - 选中状态图标(红色 #ff6b6b)

图标设计建议:
- 尺寸: 81px × 81px (小程序推荐尺寸)
- 格式: PNG (支持透明背景)
- 主题: 组织/团队相关图标,如:
  * 多人图标 👥
  * 组织架构图标
  * 团队徽章图标
  * 社团标志图标

设计规范:
- 未选中状态: 使用 #999999 灰色
- 选中状态: 使用 #ff6b6b 红色
- 保持与其他 tabbar 图标风格一致
- 线条粗细适中,识别度高

临时解决方案:
可以暂时复制 huiyuan21.png 作为占位图标,待设计师提供正式图标后替换。
